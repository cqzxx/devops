## lambda 表达式

- `lambda` 表达式用于创建简单小巧的`匿名函数`。
- 通过 lambda 表达式创建的函数不能包含语句或标注。
- 在语法上，匿名函数只能是单个表达式。
- 在语义上，它只是常规函数定义的语法糖。

代码示例：
```python
def _sum(a, b):
    return a + b

l_sum = lambda a, b: a + b

print(_sum)         # <function _sum at 0x7f590435f1f0>
print(_sum(2, 5))   # 7

print(l_sum)        # <function <lambda> at 0x7fd9d7ba3ca0>
print(l_sum(2, 5))  # 7
```

上述 `l_sum 函数`功能与 `_sum 函数`一样
