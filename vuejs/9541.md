# 常用代码片段

## 局部 css 样式（scoped 属性）
https://vue-loader.vuejs.org/zh/guide/scoped-css.html
```vue
<style scoped>// [!code focus]
.example {
  color: red;
}
</style>

<template>
  <div class="example">hi</div>
</template>
```

## 在循环中使用外部的 this
```js
this.data.forEach(function(value) {
  this.data[value.code] = value.name
}, this)// [!code focus]
```
