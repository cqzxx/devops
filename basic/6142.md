# 水仙花数

难度：:star:

## 1. 题目描述
::: tip 题目描述
- 水仙花数定义：一个`N位`非负整数，其各位数字的`N次方和`等于该数本身。
- 严格意义来说水仙花数指 3 位数。
- 在这里通过代码找出 3 位数中的水仙花数。
:::

## 2. 关键点
- pow


## 3. 代码实现
::: details 点击显示代码
```python
for i in range(100, 1000):
    str_i = str(i)
    _sum = 0
    for ii in str_i:
        _sum += pow(int(ii), 3)
    if _sum == i:
        print(i)
```
:::

## 4. 运行示例

```txt
153
370
371
407
```
